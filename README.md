## 👋 About Me

I build software that builds software.

My journey started in the .NET and TypeScript ecosystems, building APIs, SPAs, and full-stack products. Over time I moved closer to the machine — writing Rust tooling, native bindings, and automation scripts. Today I focus on **agentic development**: creating tools, proxies, and integrations that let LLMs and AI agents do real work inside real codebases.

I care about clean architecture, pragmatic design, and code that survives the next developer.

## 🧠 What I Do

- **Agentic Development & AI Tooling** — LLM integrations, OpenAI/Claude proxies, agent orchestration, prompt engineering, and automation that augments the developer workflow.
- **Backend & Architecture** — C#, .NET, Domain-Driven Design, robust APIs, and layered / n-tier architecture.
- **Frontend & Full-Stack** — TypeScript, SPAs, and seamless front-to-back integration.
- **Systems, Tooling & Automation** — Rust, Python, native bindings, debug tooling, and scripts that remove friction.
